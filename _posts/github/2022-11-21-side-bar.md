# 사이드바 메뉴 설정

## 📂categories 폴더 생성
`_pages\categories` 폴더를 만들어준다.

## 📄category.md 파일 추가
`_pages\categories`의 하위 폴더에 category-XXX.md 파일들을 생성한다.  
특정 category에 해당되는 post만 포함되도록 수정한다.

Category와 page 연결

```md
---
title: "카테고리명"
layout: archive
permalink: categories/카테고리명
author_profile: false
sidebar_main: true
---
```
