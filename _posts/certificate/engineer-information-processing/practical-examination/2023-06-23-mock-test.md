  - **표현** 계층(**Presentation** Layer) : 세션 계층 간의 주고받는 인터페이스를 일관성있게 제공

  - **전송** 계층(**Transport** Layer) : 두 호스트 시스템으로부터 발생하는 데이터의 흐름을 제공, `TCP`, `UDP` 프로토콜 사용, 데이터 단위 `세그먼트`
  - **네트워크** 계층(**Network** Layer) : 데이터를 목적지까지 가장 안전하고 빠르게 전달하는 기능(**라우팅**)을 담당, 라우터를 통해 경로를 선택하여 IP주소를 지정하고 경로에 따라 `패킷`을 전달해준다.

  - **물리** 계층(**Physical** Layer) : 데이터를 전기적인 신호로 변환해서 주고받는 기능을 진행하는 공간, `통신 케이블`, `허브`
