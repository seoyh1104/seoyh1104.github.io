## 주석
```
# TODO: 할 일
# FIXME: 고쳐야 할 부분 (치명적인 에러를 발생하는 코드는 아니지만 수정 필요)
# !!!: 경고
# ???: 의문점
# MARK: 설명
# NOTE: 노트
# BUG: 버그
# XXX: 문제가 있음 (더 생각해볼 필요가 있음)
```

## 설명
- `# TODO:` : 할 일
- `# FIXME:` : 고쳐야 할 부분 (치명적인 에러를 발생하는 코드는 아니지만 수정 필요)
- `!!!:` : 경고
- `???:` : 의문점
- `MARK:` : 설명
- `NOTE:` : 노트
- `BUG:` : 버그
- `XXX:` : 문제가 있음 (더 생각해볼 필요가 있음)
